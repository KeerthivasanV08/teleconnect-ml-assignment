# Data Directory

This folder stores all datasets used in the TeleConnect ML Assignment.

## Structure

- `raw/`  
  Contains the original raw dataset provided for the assignment.

- `processed/`  
  Contains cleaned and transformed datasets generated during preprocessing.

## Files

### Raw Data
- `raw/Telco-Customer-Churn.csv`  
  Original telecom customer churn dataset.

### Processed Data
- `processed/telco_cleaned.csv`  
  Cleaned dataset after handling missing values, datatype issues, and outliers.

Additional processed datasets for modeling will also be stored here in later stages:
- `X_classification.csv`
- `y_classification.csv`
- `X_regression.csv`
- `y_regression.csv`
